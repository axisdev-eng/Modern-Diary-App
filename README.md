<img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Rocket.png" width="35" /> 🚀 Features
🧠 TherapistChat
Interactive AI guidance to help navigate complex thoughts.

🎭 Stickers and Audio
Use custom stickers in notes based on your mood. After the mood is detected, music plays accordingly.

📋 DailyAIReport
Automated summarization of your daily notes.

⏳ Timeline View
A clean, organized history of your notes.

🎬 GIF Integration
Uses GIPHY to allow users to insert expressive GIFs.

⚙️ Profile Customization
Profile icon and bio may be changed at any time.

📷 Media Upload
You can upload and store your own images as well.

<img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Travel%20and%20Places/Vertical%20Traffic%20Light.png" width="35" /> 🚦 Running the Project
Prerequisites: Node.js

Clone the Repository

Bash
git clone https://github.com/yourusername/aura-diary.git
Install Dependencies

Bash
npm install
Configure Environment

[!IMPORTANT]
Set the GEMINI_API_KEY in .env.local to your Gemini API key.

Launch the App

Bash
npm run dev
