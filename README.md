<img src="https://cdn-icons-png.flaticon.com/512/1162/1162456.png" width="40" /> Features
<br>
🧠 TherapistChat
Interactive AI guidance to help navigate complex thoughts and emotional blocks.

🎭 Stickers and Audio
Custom stickers in notes based on your mood. Once a mood is detected, a unique soundscape plays via the Waveform Player.

📋 DailyAIReport
Automated AI-driven summarization of your daily notes to track your mental progress.

⏳ Timeline View
A beautiful, organized chronological history of every entry you've ever made.

🎬 GIF Integration
Full GIPHY support to allow for expressive, visual journaling.

⚙️ Profile Customization
Complete control over your profile icon and bio for a personalized experience.

📷 Media Upload
Support for uploading and storing your own high-quality images within notes.

<img src="https://cdn-icons-png.flaticon.com/512/2910/2910768.png" width="40" /> 🚦 Running the Project
Prerequisites: Node.js

Clone the Repository

Bash
git clone https://github.com/yourusername/aura-diary.git
Install Dependencies

Bash
npm install
Environment Setup

[!IMPORTANT]
Create a .env.local file and add your GEMINI_API_KEY.

Launch

Bash
npm run dev
